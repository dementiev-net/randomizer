# AGENTS.md

## Правила работы

- Сообщения коммитов писать с маленькой буквы.
- Формулировка коммита: глагол в 3-м лице, например `делает ...`, `добавляет ...`, `исправляет ...`.
- Один коммит должен содержать одну логическую группу изменений.

## Проверки перед коммитом

- Обязательный запуск тестов: `swift test`
- Проверка сборки приложения: `xcodebuild -project Randomizer.xcodeproj -scheme Randomizer -configuration Debug CODE_SIGNING_ALLOWED=NO build`
- Если тесты или сборка не проходят, коммит делать только после отдельного согласования.

## Git и push

- `git push` делать только по прямому согласованию с пользователем.
- Если есть только запрос на локальный коммит, остановиться после коммита и сообщить hash.
- Не использовать force-push без явного запроса.
