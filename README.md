# Randomizer

Минималистичный macOS-рандомайзер в плавающем окне.

## Что делает приложение

- Генерирует случайное число от `001` до `099` по тапу
- Автоматически обновляет число каждые 10 секунд
- Показывает нижний индикатор диапазона числа:
  - настраиваемые границы `Граница 1 / Граница 2` для диапазонов
  - пресеты в настройках: `33/66`, `25/75`, `20/80`
  - активна только одна плашка текущего диапазона
- Ведёт два таймера:
  - `Сессия` — время текущего запуска
  - `Все время` — суммарное время использования между запусками
- Подсвечивает состояние усталости по суммарному времени:
  - `warning` после 55 минут
  - `critical` после 60 минут
- Хранит настройки банкролла:
  - текущий банкролл в `$`
  - лимит шота (`NL10`, `NL25` и т.д.)
  - количество попыток шота в `BI` (используется как стоп-лосс `-N BI`)
  - порог банкролла для шота в `BI` (например, `25 BI`)
  - файл настроек: `~/Documents/Randomizer/settings.json`
- Ведёт журнал шотов:
  - добавление записи из настроек (результат в `$`, время текущей сессии, комментарий, применение к банкроллу)
  - быстрые кнопки результата в `BI` (`-1`, `-0.5`, `+1`)
  - в настройках доступны действия `Записать` и `Показать`
  - журнал открывается в отдельном окне с таблицей и графиком банкролла
  - график поддерживает режимы `По записям` и `По дням`
  - под графиком показываются `Текущий`, `Мин`, `Макс`
  - файл журнала: `~/Documents/Randomizer/shot_journal.json`
- Правило шота:
  - при суммарных `-N BI` (где `N` = значение `Попытки шота`) текущий шот автоматически закрывается
  - после восстановления банкролла до порога `M BI` (настройка `Порог шота, BI`) шот автоматически открывается снова
- Показывает готовность шота верхнего лимита по правилу порога `M BI`
  - если банкролла не хватает: `нужно $X`
  - если шот закрыт по стоп-лоссу: `ШОТ ЗАКРЫТ`
  - если банкролла достаточно: `ГОТОВ (X BI)`, где `X` — запас сверх порога в бай-инах
- В настройках показывает бюджет шота `N BI`
- Поддерживает жёсткий режим stop-loss:
  - при достижении stop-loss блокирует роллы на `N` минут
  - показывает таймер оставшегося перерыва
  - после завершения перерыва автоматически разблокирует игру и обнуляет результат сессии
- При достижении `stop-win`:
  - статус на главном экране: `stop-win достигнут`
  - новые роллы блокируются до сброса сессии
- При записи в журнал лимиты `stop-loss`/`stop-win` принудительно применяются перед сохранением записи

## Технологии

- SwiftUI
- Combine
- UserDefaults (для сохранения общего времени)
- JSON в `~/Documents/Randomizer` (для настроек банкролла)

## Запуск

1. Откройте `Randomizer.xcodeproj` в Xcode
2. Выберите схему `Randomizer`
3. Запустите приложение на `My Mac`
