# Randomizer

Генератор случайных покерных раздач с фильтрацией по условиям.

## Возможности

- 🎲 Генерация случайных раздач
- 🔍 Фильтрация по типам рук
- ⚙️ Настраиваемые параметры
- 📊 Статистика генераций

## Технологии

- SwiftUI
- Swift стандартная библиотека

## Использование

1. Выберите параметры фильтрации
2. Сгенерируйте раздачу
3. Используйте результат
