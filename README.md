# 2Dapp Randomizer

Простое macOS приложение для генерации случайных чисел с отслеживанием времени работы.

## Описание

2Dapp Randomizer — это утилита для macOS, которая периодически генерирует случайные числа и отслеживает время работы пользователя. Приложение полезно для различных сценариев, требующих случайного выбора или напоминания о времени работы.

## Возможности

- **Генерация случайных чисел**: Каждые 10 секунд генерируется случайное число от 1 до 99
- **Визуальный индикатор уровня**: Отображает уровень сгенерированного числа с помощью индикатора (0-5 уровней)
- **Отслеживание общего времени**: Показывает общее время использования приложения
- **Отслеживание времени сессии**: Показывает время текущей сессии
- **Предупреждение о длительной работе**: Меняет цвет таймера на красный после 1 часа работы
- **Сброс общего времени**: Кнопка сброса для обнуления общего счётчика времени

## Системные требования

- macOS 10.15 (Catalina) или новее
- Xcode 13.0 или новее (для сборки из исходного кода)

## Установка

### Сборка из исходного кода

1. Клонируйте репозиторий:
```bash
git clone [URL вашего репозитория]
cd 2Dapp\ Randomizer
```

2. Откройте проект в Xcode:
```bash
open "2Dapp Randomizer.xcodeproj"
```

3. Выберите схему сборки и нажмите ⌘R для запуска

## Использование

После запуска приложения вы увидите:

- **Поле с случайным числом**: Обновляется каждые 10 секунд (формат: трёхзначное число с ведущими нулями)
- **Индикатор уровня**: Визуально показывает "силу" сгенерированного числа:
  - 0: числа 1-24
  - 1: числа 25-32
  - 2: числа 33-49
  - 3: числа 50-65
  - 4: числа 66-74
  - 5: числа 75-99
- **Общее время**: Показывает общее время использования (формат: ЧЧ:ММ:СС)
- **Время сессии**: Показывает время текущей сессии
- **Кнопка Reset**: Сбрасывает общий счётчик времени

### Предупреждение о времени

Когда общее время работы превышает 1 час, цвет таймера изменится на красный, напоминая о необходимости сделать перерыв.

## Архитектура

Проект построен на базе AppKit (Cocoa) и состоит из:

- **AppDelegate.swift**: Основной делегат приложения с настройкой жизненного цикла
- **ViewController.swift**: Главный контроллер с логикой генерации чисел и отслеживания времени

### Технические детали

- Два независимых таймера:
  - Таймер генерации чисел (интервал: 10 секунд)
  - Таймер отслеживания времени (интервал: 1 секунда)
- Использование `Timer.scheduledTimer` для асинхронных операций
- Форматирование времени с помощью `DateComponentsFormatter`

```swift
let RANDINTERVAL = 10.0    // Интервал генерации чисел (секунды)
let ALERTTIME = 3600       // Время до предупреждения (секунды, по умолчанию 1 час)
```
